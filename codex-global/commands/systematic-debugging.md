# Systematic Debugging

## Objetivo

Usar a skill oficial `systematic-debugging` para investigar a causa raiz antes de propor correções.

## Instruções

1. Use a skill `systematic-debugging`.
2. Siga a skill exatamente.
3. Use `$ARGUMENTS` como o problema inicial.
4. Não proponha correções antes da investigação e hipótese.
