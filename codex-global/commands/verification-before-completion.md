# Verification Before Completion

## Objetivo

Usar a skill oficial `verification-before-completion` antes de declarar que algo está pronto, corrigido ou passando.

## Instruções

1. Use a skill `verification-before-completion`.
2. Siga a skill exatamente.
3. Use `$ARGUMENTS` para indicar o que precisa ser validado.
4. Não faça claims de sucesso sem rodar a verificação correspondente.
